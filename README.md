# Maleah-s-Playpad-
Just a toddler's fun time app.

## What changed
- Fixed the broken `index.html` markup
- Added a colorful toddler play app interface
- Added PWA support with a manifest and service worker
- Added iPhone home screen install guidance inside the app

## iPhone install
1. Open the deployed site in Safari.
2. Tap the **Share** button.
3. Tap **Add to Home Screen**.
4. You can then open it like an app or share the link with your daughter.
